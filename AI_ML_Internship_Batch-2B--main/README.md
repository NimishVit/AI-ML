**Adult Census Income Prediction**

This project implements an end-to-end machine learning pipeline to classify individual income levels based on demographic data. The workflow encompasses data imputation, categorical encoding, feature scaling, and the training of multiple classification algorithms including Logistic Regression, Decision Trees, Random Forests, K-Nearest Neighbors, and Support Vector Machines. Model performance is rigorously evaluated using Accuracy, Precision, Recall, F1-Score, and ROC-AUC metrics to ensure robust predictive capabilities.

**Live Face Detection**

Engineered for real-time computer vision applications, this project detects human faces through a live webcam feed. It processes video frames dynamically, applying bounding boxes around detected faces with minimal latency. The implementation focuses on highly optimized inference speeds, showcasing methodologies suitable for edge-device integration and privacy-preserving local execution.

**Cancer Detection**

A predictive medical AI model designed to classify benign and malignant tumors based on clinical diagnostic datasets. The project addresses the complexities of working with biological data, particularly focusing on mitigating class imbalance and prioritizing recall metrics to minimize false negatives in critical diagnostic scenarios.

**Image Classification**

This application leverages deep learning techniques to categorize images into distinct classes. The model processes pixel data through optimized architectures to extract spatial hierarchies and visual features. It demonstrates proficiency in handling image preprocessing, tensor manipulation, and optimizing neural network parameters to achieve high classification accuracy on unseen validation data.
